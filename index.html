Timeline

5주 개발 일정

기획부터 데모까지 5주 안에 완성합니다.

1
Week 1
기반 세팅 & 파이프라인 설계
  • FastAPI 서버, Redis, DB 세팅
  • ffmpeg 프레임 추출 파이프라인 구축
  • Claude API 연동 테스트
2
Week 2
AI 핵심 기능 개발
  • Claude Vision으로 방송 프레임 제품 인식
  • 네이버쇼핑 API 상품 자동 매칭
  • 인식 정확도 테스트 및 튜닝
3
Week 3
웹앱 자동생성 & 배포
  • Claude API로 쇼핑 웹앱 HTML 자동 생성
  • 생성된 웹앱 서버 자동 배포
  • 관리자 대시보드 개발
4
Week 4
TV UI & 통합
  • 셋탑 팝업/배너 UI 개발
  • 리모컨 네비게이션 구현
  • 홈화면 → 쇼핑 웹앱 전환 플로우 완성
5
Week 5
통합 테스트 & 데모 준비
  • 전체 파이프라인 E2E 테스트
  • 발표용 데모 시나리오 구성
  • 성과 지표 시뮬레이션
